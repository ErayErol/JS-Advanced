Ние трябва да разделиме 3-те основни елемента в нашето приложение
1. Презентационна логика => нещо което комуникира с потребителя(клиента)
2. Биснез логика => освноната функционалност на нашето приложение
3. Работата с данни/инфраструктурата
4. Тест

Биснез логиката трябва да е независима, не трябва да е обвързана с Презентационна логика,
защото ние можем да сменим Презентационна логика 
=> Пример: Днес ще бъде конзолно приложение, утре Уеб страница, други ден Мобилно приложение

Правим dependency 
=> От класа където започва Program dependency => Core (Биснез логиката)
От Core (Биснез логиката) => Инфраструктурата
