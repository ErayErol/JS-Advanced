1. Клиента праща => request
2. Сървъра праща => response
3. Този response е текс
4. Браузъра парсва текста и създава WEB страница от него 
5. Ключовата част от тази страница е => DOM
6. DOM е едно дърво

DOM
=> API, което е било предоставено от браузъра, за да може да контролираме WEB страница
=> Обекти които репрезентират текстово (HTML) съдържание
=> JS посредством DOM можем да променяме съдържанието на нашата страница по динамичен начин
=> се достъпва с document

Атрибут и Пропърти
Едно е да зададем свойства на обекта на обекта в паметта
Друго е да зададем атрибут на HTML елемент
