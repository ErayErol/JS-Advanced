Model	
=> класове, които описват данните
=> всеки един клас който репрезентира някакъв реален обект от живота 

View 
=> презентиране на информацията за потребителя
=> user interface (презентация пред потребителя)

Controller
=> описва как данните от Model да се прехвърлят във (html) View
=> съдържа бизнес логиката вътре
=> контролера е клас и action e метод в този клас 

Router
=> знае как да ротира URL-лите и да ги насочи към правилните action-и и към правилните контролери
